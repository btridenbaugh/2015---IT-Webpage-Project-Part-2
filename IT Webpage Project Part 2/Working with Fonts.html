Working with Fonts

There are many ways to work with fonts in html. Some of the basic ways, are through the use of bold, underline, italics, and oblique, you can create superscripts, and subscripts. You can also effect the font properties. You can make it small or large. You can change the font family used in the text. Also you can change the color of the text using things like red, yellow, and green. Also if you would like a specific font color throught the use of color hex codes you can do that too.

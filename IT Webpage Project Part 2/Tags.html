Tags I've Learned
< html > Defines an HTML Document
< title > Defines the Title of the Document
< head > Where Style Sheets and other files are Inserted
< body > Defines the Body of the Document
< h1 >...< h6 > All Heading Types
< br > Inserts a Line Break
< link > Used to insert Style Sheets or other outside styles
< ul > Creates an Unordered list
< ol > Creates an Ordered list
< li > Creates a list item
< div > Region to be formatted
< small > Small text
< big > Big text
< sup > Superscript
< sub > Subscript
< i > Italics
< b > Bold
< o > Oblique
< tt > Monospaced text
< table > Creates a table
< tr > Creates a table row
< th > Defines a table heading
< td > Defines a table data cell
< a > Used to insert a Hyperlink
< font > Used to configure font properties within the html
< div > A container for an element
< center > Centers the element it is applied to
